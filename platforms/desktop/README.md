# Desktop (Flutter)

## Dependências Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install -y clang cmake ninja-build pkg-config libgtk-3-dev \
  libgstreamer1.0-dev libgstreamer-plugins-base1.0-dev
```

## Linux
```bash
bash platforms/desktop/run_linux.sh
```
