# Web (Flutter)

Origem real: `frontend_flutter/`

## Rodar web local
```bash
bash platforms/web/run.sh
```

## Build web release (produção)
```bash
bash platforms/web/build_release.sh
```

Opcional com API custom:
```bash
NOVA_API_URL=https://sua-api.exemplo.com bash platforms/web/build_release.sh
```
